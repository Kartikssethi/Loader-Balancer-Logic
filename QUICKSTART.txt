🔧 TROUBLESHOOTING:

❌ Problem: "Connection refused" error
✅ Solution: 
   - Make sure all 4 terminals have the services running
   - Wait 3 seconds between starting load balancer and opening GUI
   - Check that no other service is using ports 8000-8003

❌ Problem: Requests keep failing
✅ Solution:
   - Make sure backend servers are running
   - Try clicking "Clear Log" to reset
   - Refresh the browser page

❌ Problem: Can't open browser automatically
✅ Solution:
   - Manually open: http://127.0.0.1:8000
   - Make sure port 8000 shows "Tatkal" in page title
